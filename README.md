# CLI typing trainer

* build with `go build .` or `go install .`

* Modes supported:

    + Random words mode: `./typing words [N]`, where N is the number
    of words (defaults to 20)

* Controls:
    
    + `Ctrl+C` to display stats and quit.

    + `Ctrl+R` to display stats and restart.

* `./typing help` for help.
